##  Conceptos Aplicados

- **Búsqueda lineal:** recorre todo el arreglo comparando elemento por elemento hasta encontrar el valor buscado.  
- **Ciclo `for` y `while`:** permiten recorrer estructuras de datos de distintas formas.  
- **Arreglos unidimensionales:** estructura básica para almacenar y acceder a varios valores del mismo tipo.  
- **Retorno de valores:** uso de `int` para índices y `Integer` para manejar valores nulos (`null`).  
- **Modularidad:** separación del código en paquetes (`controllers`, `views`) para una mejor organización.
